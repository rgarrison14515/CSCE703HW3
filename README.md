# OWASP Login Form Assignment

This project is a simple login form created for a cybersecurity risk assignment. It demonstrates basic frontend and backend validation techniques.

## Features

- Email and password input fields
- Client-side validation using JavaScript
- Server-side validation using Node.js and Express
- Basic error handling and user feedback

## Validation Rules

- Email must not be empty and must contain "@"
- Password must not be empty and must be at least 8 characters long

## Technologies Used

- HTML
- JavaScript
- Node.js
- Express

## How to Run

1. Install dependencies:
npm install
2. Start the server:
node server.js
3. Open the application in your browser:
4. http://localhost:3000


## Notes

This project is intentionally simple and is designed to demonstrate input validation and basic web security concepts.
